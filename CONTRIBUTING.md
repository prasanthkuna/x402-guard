# Contributing

## Setup

```powershell
npm install
npm run build
npm run test
```

## Pull requests

1. One concern per PR (policy change, middleware API, Go port, docs).
2. Include tests for behavior changes.
3. Match existing style: strict TypeScript, minimal comments, fail-closed defaults.
